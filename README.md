**Godot-Fetch** — это абсолютно ненормальный, оверкильный и ультра-кастомный системный фетч, написанный на игровом движке **Godot Engine 4.6+** для **CachyOS / Arch Linux**. 

Зачем использовать легковесный `fastfetch` на C, если можно запустить целый графический 3D/2D движок, инициализировать Vulkan API, залупить кастомный парсер шейдеров, прогнать регулярки через костыли и выдать ASCII-пингвина в термукс с задержкой, которая заставит твой процессор плакать? **Потому что мы можем.**
